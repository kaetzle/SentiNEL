
1. Installer nodejs 
	- voir lien: "How to Install Node.js® and NPM on Windows - Treehouse Blog")

2. Aller dans le répertoire contenant les fichiers package.json & sentiments.js

3. Charger les modules nécéssaires au fonctionnement du programme
	- commande
		npm install

4. Lancer le programme
	- commande
		node sentiments.js Restaurant-Train-SB1-2016.xml

5. 2 fichiers sont créés contenant l'extraction des données du fichier xml
	- sentiments.csv
	- sentiments.json
